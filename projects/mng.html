Measurement Network Gateway

Measurement Network Gateway collects data from multiple sensor nodes and routes it to a user interface to inform users about the system. Sensor data is acquired by the Programmable Logic (FPGA) due to its high I/O capability and parallel sampling, while the Processing System (ARM Cortex-A9) is used for networking and control management. On the Zynq platform, symmetric multiprocessing (SMP) is utilized. Linux is used as the operating system.

What?

  • SW/HW co-design for sampling sensor nodes with the PL, and control management and networking with the PS using Embedded Linux.
  • The core objective is to create reliable networking between the SoC and the GUI, enabling communication for controlling sensor sampling frequencies, managing sensors, and performing read/write operations via the AXI interface.

How?

  • Implemented SPI in VHDL using Vivado.
  • Created AXI IP to control hardware from the PS side.
  • Embedded Linux image created using PetaLinux.
  • GUI developed using the GTK library.
